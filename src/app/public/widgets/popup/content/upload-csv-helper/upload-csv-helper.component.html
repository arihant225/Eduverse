Integrate new users seamlessly by uploading a CSV file containing their information. Ensure a smooth onboarding process for enhanced user management.

Instructions:

  1. CSV Format Requirements:

    Ensure your CSV file follows the specified format for successful integration. Include columns such as:

    • First Name
    • Last Name
    • Email
    • USER Id [optional] (if needed Cutom)

  2. Data Validation:

    Double-check the accuracy of the data in your CSV file to prevent errors during integration. Verify email addresses, remove duplicates, and confirm the correctness of other user details.

  3. File Upload:

    Use the file upload button below to select and upload your CSV file. Supported file formats: CSV (Comma Separated Values)

  4. Integration Settings:

    Specify any additional integration settings or preferences before initiating the process. Choose default user roles, set permissions, or configure other relevant options.

  5. Review and Confirm:

    After uploading, review the summary of the data to be integrated. Confirm that the information is accurate and matches your expectations.

  6. Initiate Integration:

    Click the "Integrate Users" button to start the integration process. Monitor the progress and receive confirmation upon successful completion.

Additional Notes:

  • For any assistance or troubleshooting, refer to the provided documentation or contact support.
  • Ensure a stable internet connection during the upload and integration process.
  • Large files may take some time; please be patient.

Important: By proceeding, you acknowledge that incorrect or incomplete data in the CSV file may lead to errors in user integration.
